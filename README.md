## 1. 项目说明
 【net】音乐播放器

## 2. 项目所需控件

|控件名|说明|
|:-:|:-:|
|Form|窗体|
|groupBox|实现控件分组|
|imageList|存放列表图标|
|tableLayoutPanel|美化按钮组|
|listView|显示歌单和歌曲信息|
|label|提醒+说明|
|pictureBox|放置Logo|
|textBox|输入框|
|button|按钮|

## 3. 项目开发及运行环境
|名称|版本|
|:-:|:-:|
|Windows|7.0|
|Visual Studio|2012|
|.NET Framework|4.5.2|
|MySQL|8.0.19|

## 4. 功能实现
### 4.1 用户信息处理
- 用户注册
- 用户登录
### 4.2 歌曲信息处理
- 获取热门歌单
- 获取推荐歌单
- 获取歌曲信息
- 获取用户收藏歌单
### 4.3 用户操作
- 新建歌单
- 删除歌单
- 添加音乐到歌单
- 从歌单中删除音乐
- 播放歌曲
- 上一曲/下一曲
- 随机切歌
